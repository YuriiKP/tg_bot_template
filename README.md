# Telegram Bot Template

Шаблон Telegram бота с админ-панелью, ролевой системой пользователей и рассылкой сообщений. Использует базу данных SQLite (или PostgreSQL)

## Функции

- Админ-панель для управления ботом
- Ролевая система пользователей
- Рассылка сообщений всем пользователям
- Интеграция с базой данных

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url>
   cd tg_bot_template
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Настройте переменные окружения в `.env` файле (токен бота и т.д.).

## Запуск

### Локально
```bash
python main.py
```

### С Docker
```bash
docker-compose up --build
```

## Структура проекта

- `main.py` - точка входа
- `handlers/` - обработчики команд и сообщений
- `keyboards/` - клавиатуры для бота
- `utils/` - вспомогательные функции
- `storage.py` - работа с базой данных
- `filters.py` - фильтры для сообщений
- `commands.py` - команды бота
